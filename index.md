# ğŸ§­ 3. Loading datasets into R / Python

Example using **BOD.csv**:

### R
```r
url <- "https://pmking123.github.io/statistics-data/datasets/BOD.csv"
dat <- read.csv(url)
```

or

```r
library(readr)
url <- "https://pmking123.github.io/statistics-data/datasets/BOD.csv"
dat <- read_csv(url)
```

### Python
```python
import pandas as pd
df = pd.read_csv("https://pmking123.github.io/statistics-data/datasets/BOD.csv")
```
